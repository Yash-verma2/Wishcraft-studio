# WishCraft Viral Reel Studio & Agent 💀✨

Welcome to the **WishCraft Viral Reel Studio & Agent**! This is a state-of-the-art, premium web application built to help creators, marketers, and social media growth managers design, simulate, and export high-viral-potential Reels for **WishCraft** (the ultimate digital magical-experience gifting platform).

Designed with a sleek, glowing, dark glassmorphic interface, this tool features an interactive 9:16 mobile phone simulator that previews animated, high-fidelity social media reels complete with simulated chat messages, modern audio sync, particle effects, and customizable transition modes.

---

## 🌟 Key Features

1. **AI Daily Viral Ideas Agent**: 
   * Pre-loaded with highly optimized viral video hooks, recommended trending soundtracks, visual storyboards, and high-converting captions customized for WishCraft's organic growth.
   * "Load into Studio" button instantly configures the entire workspace and simulator.

2. **Interactive 9:16 Smartphone Simulator**:
   * **Scene 1 (The Hook - Relatable Chat)**: Simulates a real-time messaging thread (avatar, usernames, and double-clickable typing bubbles with typing animations).
   * **The Pivot (Custom Transitions)**: Choose from 3 high-impact styles: *Magical Sparkle Explosion*, *Vibrant Glitch Blur*, or *Cinematic Zoom Blur*.
   * **Scene 2 (The Reveal - Magical Website)**: Simulates a real WishCraft landing page featuring customizable theme gradients (Starry Midnight, Pastel Sweetheart, Sunset Glow, Neon Cyberpunk) and active floating particles (Hearts, Balloons, Stars).
   * **Social HUD Overlay**: Renders authentic Instagram/TikTok interface overlays (like count, comment count, share, follow banner, and custom captions).

3. **Graphic Asset Exporters**:
   * Uses built-in custom 2D Canvas renderers to generate and export crisp, high-definition PNGs of both the simulated chat thread and the magical WishCraft landing card. 
   * Perfect for creating instant thumbnails, story sliders, or multi-photo carousel posts!

4. **Synchronized Simulated Sounds**:
   * Interactive acoustic assets that play typewriter typing sounds, glitch effects, card-reveal swooshes, and magical chime sweeps synced perfectly with the animation timeline.

## 🚀 How to Run the App Instantly

The project is completely self-contained with no heavy framework compilation or installation required. You can run it instantly using any of the methods below:

### Method 1: Double-Click (Easiest)
Simply open your file manager, navigate to this workspace, and double-click `index.html` to open it directly in any modern web browser (Chrome, Safari, Edge, Firefox).

### Method 2: Python Local Dev Server
If you want to run it on a local server, open your terminal in this workspace directory and run:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.

### Method 3: Node.js / npx Server
If you prefer Node.js, run:
```bash
npx serve .
```
Then open the port displayed in your terminal (usually `http://localhost:3000`).
